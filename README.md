---  LOOP projekt  ---

Ebben a könyvtárban van minden ami a Loop.
Iskolai esemény naptár egy kis csavarral.

Fő funkciók: 
-Iskola esemény naptár, helyszínnel, értékelési rendszerrel
-Iskolai diákönkormányzatnak magas szintű HR megoldás
-Szerevezés lebonyolítás, gyors infomáció áramlás, személyes teljesítmény profil

Mértéktelen IDÖ-s aktivitás után az ötlet magáért beszél. Nem kiforrott rendszer egy iskola diákélet működtetésére. Ennek itt vége.
